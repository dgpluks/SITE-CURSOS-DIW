CURSOS-MG
O curso de Computação, também chamado de Ciência da Computação, é uma graduação que prepara profissionais para o desenvolvimento de sistemas computacionais.

Conteúdo do curso

  • Estudo de algoritmos,sistemas de software,hardware e redes de computadores
  • Análise matemática de algoritmos e complexidade computacional
  • Programação de software
  • Administração de sistemas de computador
  • Segurança da informação
  • Engenharia de software
  • Sistemas distribuídos

Formação

  • O curso de Computação pode ser do tipo bacharelado ou licenciatura
  • É possível cursar a graduação nas modalidades presencial e a distância
  • Os alunos têm acesso a laboratórios equipados, onde podem aplicar os conceitos teóricos em aulas práticas

Oportunidades de carreira

  • O curso de Computação prepara os alunos para oportunidades no setor tecnológico
  • Os alunos podem se candidatar a bolsas de Iniciação Científica para desenvolver atividades de pesquisas

Oportunidades de carreira

  • Desenvolvimento, implementação e gestão de sistemas e soluções computacionais
  • Análise de sistemas
  • Gestão de redes de dados
  • Segurança da informação
